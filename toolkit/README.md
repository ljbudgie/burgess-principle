# Citizen's Legal Toolkit

This directory contains ready-to-use templates for deploying the Burgess Principle framework against any institution. Each template is copy-paste ready — fill in the `[PLACEHOLDER]` fields, send it, and create your evidential record.

No legal training is required. No fee is charged.

For step-by-step guidance on how to use these templates, see the [Tutorials](../tutorials/README.md).

---

## Templates

| Template | Description |
|----------|-------------|
| [General Subject Access Request](TEMPLATE_SAR_GENERAL.md) | A professional SAR template that works for any institution, covering all personal data, automated decisions, and third-party sharing. |
| [Warrant-Specific SAR](TEMPLATE_SAR_WARRANT.md) | A forensic SAR template for challenging warrants, requesting the warrant document, court application, magistrate details, and metadata. |
| [Enforcement Visit SAR](TEMPLATE_SAR_ENFORCEMENT.md) | For challenging any enforcement visit — bailiffs, debt collectors, energy agents — requesting authorisation records, bodycam footage, and vulnerability flags. |
| [Automated Decision-Making Challenge](TEMPLATE_ARTICLE22_CHALLENGE.md) | A standalone Article 22 UK GDPR challenge for use when a decision was made by an automated system without meaningful human intervention. |
| [Reasonable Adjustment Assertion](TEMPLATE_EQUALITY_ACT.md) | A template for asserting reasonable adjustments under section 20 of the Equality Act 2010, ensuring all communication is accessible and documented. |
| [Ombudsman/Regulator Complaint](TEMPLATE_OMBUDSMAN_REFERRAL.md) | For escalating to the relevant ombudsman or regulator when an institution fails to respond or fails to answer the question. |
| [Court Application Guide](TEMPLATE_COURT_APPLICATION.md) | A practical guide explaining when and how to issue court proceedings, including claim forms, fees, and what to include in your case. |

## How to Use

1. **Choose your template** — pick the one that matches your situation.
2. **Copy the template** — each template contains a clearly marked email or letter you can copy directly.
3. **Fill in the placeholders** — replace every `[PLACEHOLDER]` with your own details.
4. **Send it** — by email where possible, so everything is timestamped and documented.
5. **Record the date you sent it** — the 30-day clock starts from the date of receipt.
6. **If they don't respond** — use the [Ombudsman/Regulator Complaint](TEMPLATE_OMBUDSMAN_REFERRAL.md) template to escalate.
